## Project Philosophy

This project is designed to be a **standard application shell** for building Docker-ready Node.js services within the News Workspace ecosystem. It focuses on stability, consistency, and developer experience.

**We value your contributions!** However, to keep the core library consistent, we ask that you **please open an issue to discuss significant changes before submitting a Pull Request**.

## Code of Conduct

By participating in this project, you agree to maintain a respectful and inclusive environment for everyone as described in our `CODE_OF_CONDUCT.md`.
